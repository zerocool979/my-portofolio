About Me

I am Bilal, a 2nd semester Informatics Engineering student who has a deep interest in cybersecurity, especially Red Team Operations, penetration testing, and bug bounty. I started my journey autodidactically, exploring various topics ranging from networking, exploit development, bypass authentication, to Active Directory exploitation.

In my learning process, I have experimented directly with various tools such as Proxychains, Nmap, Burp Suite, Metasploit, and various Windows exploitation methods such as SMB brute force, Kerberos attacks, LSASS exploit, and lateral movement techniques. I also developed an interest in AI development and humanoid robot projects, where I built a client-server system and sensor-based control as a background service on Android.

I highly value hard work, technical exploration, and consistency in learning. In every project, I always try to combine technical and philosophical approaches so that the resulting solution is not only technically strong, but also humanly relevant. Outside of the technical field, I am also active in digital campaign design, video clip production, and creating educational and historical content based on AI.

My vision for the future is to become a Red Team specialist who is able to create sophisticated attack simulations, contribute to improving organizational cyber defenses, and build tools and frameworks that are useful for the ethical hacker community in Indonesia.

My Latest Project

AI Gesture Control For Robot Humanoid

Python Programming | AI (Artificial Intellegence) | Computer Vision

This project is part of my experiment in developing gesture-based humanoid robot control using Python Programming, AI and computer vision. The system uses a hand tracking model (MediaPipe) to recognize hand movements in real-time through the camera, which is then translated into robot control commands. The focus of this project is on creating a lightweight gesture recognition system, can be run locally (without cloud), and can be further developed as a remote control for robots controlled by IMU sensors.
